# ChatGPT Csomag Leírás

Ez a csomag a Tüskevár DSE projekthez összeválogatott tartalmi és szerkezeti fájlokat tartalmazza.

Cél:
- professzionális magyar webes szöveg készítése
- oldalszintű tartalmi finomítás
- CTA, SEO, trust copy és AI asszisztens dialógusok fejlesztése

A csomag szándékosan nem tartalmaz:
- `node_modules/`
- `.next/`
- build artifactokat
- runtime logokat
- érzékeny vagy futás közbeni adatállományokat

Ajánlott használat:
1. Töltsd fel ezt a csomagot ChatGPT-be.
2. Add mellé a `CHATGPT_CONTENT_PROMPT_HU.md` fájl teljes tartalmát.
3. Kérj tőle kész szövegverziókat, ne csak ötleteket.
4. Az elkészült tartalmat utána vissza lehet építeni a projekt `content/`, `app/` és `components/` fájljaiba.
