Google Calendar eseménykezelés Codexből

Ez a dokumentum azt írja le, hogyan lehet Codexből lokálisan eseményeket létrehozni, módosítani, listázni és törölni a Tüskevár DSE nyilvános Google Calendar naptárában.

1. Működési modell

A honlap továbbra is statikus marad. A Google Calendar események nem GitHub commitként kerülnek be, hanem közvetlenül a Google Calendarba. A honlap az eseményeket a beágyazott naptárból mutatja. 

Codex parancs
  -> lokális Node.js script
  -> Google Calendar API
  -> Google Calendar esemény
  -> honlap beágyazott naptára

2. Jogosultsági modell

A script service account kulccsal működik. A kulcs nem kerülhet GitHubra. A repositoryban csak a script van, a titkos JSON fájl a .secrets mappába kerül, amit a .gitignore kizár.

A Google Calendar API esemény létrehozásához jogosultság kell; az esemény létrehozás, módosítás és törlés hivatalos Calendar API művelet. Forrás: Events insert.

3. Google oldali előkészítés

  1. Google Cloud Console-ban létre kell hozni egy projektet.
  2. Engedélyezni kell a Google Calendar API-t.
  3. Létre kell hozni egy service accountot.
  4. A service accounthoz JSON kulcsot kell generálni.
  5. A JSON fájlt ide kell menteni lokálisan: .secrets/google-calendar-service-account.json.
  6. A Google Calendarban meg kell osztani a naptárat a service account email címével.
  7. A jogosultság legyen: események módosítása.

4. Parancsok

Események listázása:

npm run calendar -- list --from 2026-06-01 --to 2026-07-01

Új esemény létrehozása:

npm run calendar -- create --title "Nyílt sportnap" --date 2026-06-20 --start 15:00 --end 17:00 --location "Tüskevár iskolaudvar" --description "Nyilvános program."

Egész napos esemény:

npm run calendar -- create --title "Családi sportnap" --date 2026-06-20 --all-day --location "Tüskevár iskolaudvar"

Esemény módosítása:

npm run calendar -- update --id EVENT_ID --title "Új cím" --date 2026-06-21 --start 16:00 --end 18:00

Esemény törlése:

npm run calendar -- delete --id EVENT_ID

5. Hogyan használható telefonról Codexszel?

A felhasználó elég, ha természetes nyelven megadja:

Tegyél be egy eseményt:
Nyílt sportági bemutató
2026. június 20.
15:00-17:00
Tüskevár iskolaudvar
Leírás: labdarúgás és kosárlabda bemutató új érdeklődőknek.

Codex ebből elkészíti és lefuttatja a megfelelő npm run calendar parancsot. Törléshez előbb listázni kell az eseményeket, majd az event ID alapján lehet törölni.

6. Biztonsági szabályok

  • A .secrets mappa nem kerülhet GitHubra.
  • Csak a publikus programnaptárhoz adjatok jogosultságot a service accountnak.
  • Belső vagy érzékeny eseményekhez külön naptár és külön jogosultsági rend kell.
  • Törlés előtt mindig listázni kell, hogy biztosan a megfelelő esemény ID kerüljön törlésre.
  • Ha a kulcs kiszivárog, a Google Cloud Console-ban azonnal törölni kell.
