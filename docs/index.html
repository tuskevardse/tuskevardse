Tüskevár DSE honlap üzemeltetési dokumentáció

Ez a dokumentáció azt írja le, hogyan épül fel és hogyan tartható életben a Tüskevár DSE honlap ZTS domain regisztrátor, Cloudflare, GitHub, Codex és lokális fejlesztőkörnyezet használatával.

Dokumentumok

01 - Rendszerarchitektúra és 0 Ft-os működési modell

Áttekintés arról, melyik szolgáltatás mit csinál, miért nincs szükség saját webszerverre, és mitől marad havi 0 Ft a működés.

02 - ZTS domain és Cloudflare DNS beállítás

Lépésről lépésre: domain regisztrátor, névszerver átállítás, DNS rekordok, email rekordok, Worker custom domain.

03 - GitHub és Codex fejlesztési folyamat

Repository, jogosultságok, token/credential kezelés, commit/push, Codex szerepe és fejlesztési szabályok.

04 - Lokális tesztelés és produkcióba vitel

npm, dev server, lint, build, statikus export, Cloudflare automatikus deploy, hibakeresési sorrend.

05 - Karbantartási checklisták

Rutin műveletek, tartalmi módosítás, domain/email ellenőrzés, rollback és kritikus pontok.

06 - Google Calendar eseménykezelés Codexből

Lokális script események létrehozására, módosítására, listázására és törlésére Google Calendar API-val.

Google Workspace nonprofit egyesület számára

Google for Nonprofits, Workspace aktiválás, Gmail saját domainnel, DNS rekordok, 0 Ft-os integráció.

Rövid működési lánc

A napi fejlesztési folyamat: Codex lokálisan módosít → npm run lint → npm run build → git commit → git push → Cloudflare automatikusan buildel és publikál.
